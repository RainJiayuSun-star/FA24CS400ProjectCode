Red-Black Tree Insert Algorithm

Red-Black Tree Properties Review

Properties

  • Colors: every node is either red or black
  • Restriction: a red node cannot have a red child node
  • Balance: the black height is consistent across the tree
  • Bonus: the root node should always be black

An Overview of the Insertion Process

Key steps

  • Insert the node as in a Binary Search Tree.
  • Color the new node red.
  • Fix violations (if there is one) using rotation and recoloring.
